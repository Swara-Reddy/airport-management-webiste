# SkyLink Airport Management Website

A modern, premium-style airport web application inspired by world-class airports like Changi. This project provides an interactive and user-friendly interface for passengers to explore flights, dining, shopping, experiences, and support services.

---

##  Features

###  Fly Section

* Explore destinations with a smooth auto-scrolling city showcase
* Interactive city cards with dynamic navigation
* Premium UI with seamless animations

###  Dine & Shop

* 10+ Dining options and 8+ Shopping outlets
* Advanced filters:

  * Location (Terminal/Jewel)
  * Airport Area (Public/Transit)
  * Category (Cafe, Fashion, Spa, Gaming, etc.)
* Clickable cards → Detailed premium pages
* Realistic airport-style UI

###  Experience

* Explore airport attractions and activities
* Filter-based discovery system
* Clean, modern card layout inspired by real airport websites

###  Help & Support

* Baggage tracking information
* Lost & Found services
* Special assistance support
* Contact details (email, phone, help desk)
* Airport map integration (separate page)

###  Authentication System

* Login with validation
* Multi-step signup flow
* Profile completion form (premium UI)
* LocalStorage-based session handling
* Personalized welcome message

---

##  Tech Stack

* **CSS3** – Styling (Premium UI, animations, gradients)
* **JavaScript (Vanilla)** – Interactivity & logic
